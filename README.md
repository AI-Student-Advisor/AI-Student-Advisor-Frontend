# Front-End Component README

## Environment

| Requirement | Version Specification |
|-------------|-----------------------|
| node.js     | `^20`                 |
| pnpm        | Latest                |

## Usage

### Initialization

```shell
pnpm i
```

### Development Server

```shell
pnpm dev
```

### Build Distributable

```shell
pnpm build
```

### Lint

```shell
pnpm lint
```

### Format

```shell
pnpm format
```

## Note

Lint and format before pushing.
